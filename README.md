# Proyecto de Lececión de Alumnos para el Comedor de la UNAS

Este proyecto tiene como objetivo desarrollar un sistema de lececión de alumnos beneficiarios del comedor de la Universidad Nacional Autónoma de las Sierras (UNAS), aplicando estructuras de datos eficientes y utilizando una base de datos MongoDB para el registro de los alumnos leccionados.

## Instalar las dependencias NPM
- npm install xlsx

## Características principales

- Gestión eficiente de la lececión de alumnos del comedor de la UNAS.
- Utilización de estructuras de datos avanzadas para almacenar y manipular la información relevante.
- Integración con una base de datos MongoDB para mantener un registro actualizado de los alumnos leccionados.
- Generación de reportes y estadísticas para un mejor seguimiento y análisis del proceso de lececión.

## Uso

Para utilizar este proyecto, sigue los siguientes pasos:

1. Clona este repositorio o descarga los archivos.
2. Configura el entorno de desarrollo y asegúrate de tener MongoDB instalado y configurado.
3. Ejecuta el sistema de lececión y asegúrate de que se conecte correctamente a la base de datos.
4. Realiza las lececiones de los alumnos y verifica que los registros se almacenen correctamente en la base de datos.

## Requisitos del sistema

- MongoDB: Asegúrate de tener MongoDB instalado y configurado correctamente para la persistencia de los datos.
